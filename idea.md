Project Title Agri Price Checker – Real-time Market Price App for Farmers

Your Name & Roll Number Siddu Hamigi – 2024-B-08112005A

Problem Statement Farmers often sell their produce at lower prices due to a lack of real-time information about current market rates. This results in reduced profits and missed opportunities. A simple, accessible tool is needed to provide updated crop prices so farmers can make informed selling decisions.

Proposed Solution / Idea A React Native mobile app that allows users to quickly check the latest prices of various crops in nearby markets. Farmers can search for crops, mark favorites for faster access, and view price trends over the past week. This will help them decide the best time and place to sell their produce.

Key Features

Crop search and selection with images

Display of today’s price with market details

Option to save favorite crops for quick access

Last 7 days price trend chart (optional)

Support for local languages

Works offline with previously fetched data

Target Users / Audience Farmers, agricultural traders, and small market vendors.

Technology Stack

Frontend: React Native

Storage: AsyncStorage (for favorites)

Data Source: Static JSON or simple price API

Navigation: React Navigation

Optional: Chart library for price trends, Google Maps API for market location

Expected Outcome A functional mobile app that provides farmers with quick access to market prices of selected crops, helping them maximize profits and plan sales effectively.
